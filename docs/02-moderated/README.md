# Moderated Content

Moderated content requires qualified review before it becomes canonical or appears as authoritative application material.

## Requires Moderation

- Study books, courses, professional learning paths, and academic reference material.
- Legal content, policies, governance proposals, compliance material, and contracts.
- Material tied to user certification or professional authority.
- System operations documentation that can affect deployment, security, or production behavior.

## Reviewer Rule

A moderator can route and enforce process. A certified user can contribute or review only where their verified profession matches the content category rule. Admin, founder, and system roles can manage the full workflow.
