# Intake

Use this folder for drafts, imported documents, and material that has not yet been assigned a durable category. Nothing here should be treated as canonical until it receives classification metadata and moves into `01-unmoderated/` or `02-moderated/`.

## Intake Checklist

- Identify the intended audience.
- Identify whether the document makes professional, legal, policy, academic, financial, medical, safety, or governance claims.
- Add the metadata block from `templates/content-frontmatter.md`.
- Run or apply the app's content classifier.
- Move the content into the lane and category folder selected by the classifier.
